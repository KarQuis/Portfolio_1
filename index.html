Game Score

Aplicación web para la evaluación de videojuegos, a través de una valoración definida en puntuación y comentarios.
Es posible acceder a la información de los videojuegos sin registrarse, sin embargo, para evaluar un videojuego es necesario realizar un registro de usuario con un formulario simple.
